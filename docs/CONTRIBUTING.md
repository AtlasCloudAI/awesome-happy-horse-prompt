# Contributing to Awesome Happy Horse 1 Prompts

## How to Contribute

We accept prompt submissions through GitHub Issues so the repository keeps a consistent format.

1. Open the `Submit Prompt` form.
2. Fill in the title, prompt, category, preview video URL, and source information.
3. Wait for review.
4. After the team adds the `approved` label, the workflow syncs the prompt into `data/prompts.json` and regenerates all README files.

## Submission Guidelines

- Keep the title clear and short.
- Provide the full Happy Horse 1 prompt text.
- Add a short description explaining what the clip demonstrates.
- Include a public MP4 preview URL when possible.
- Credit the original author and source link.
- Use one of the existing categories so the dataset stays consistent.

## Accepted Categories

- 超逼真视频生成
- 角色与场景一致性
- 高级运镜动作
- 创意视觉特效
- 剧情发展与延伸
