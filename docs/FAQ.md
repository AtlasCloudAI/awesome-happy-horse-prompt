# Frequently Asked Questions

## What is this repository?

This repository is a curated collection of Happy Horse 1 prompts derived from the AtlasCloud prompt ecosystem. It mirrors the structure of the GPT Image 2 awesome repository but focuses on video prompts and MP4 previews.

## How often is README updated?

The README files are regenerated by GitHub Actions on schedule and after approved submissions are synced.

## Can I submit prompts in other languages?

Yes. The dataset stores the original prompt language, and the repository publishes multi-language README wrappers for broader browsing.

## Does this repository store video files?

No. The repository stores prompt metadata and external preview URLs.

## How do I report a problem?

Open a GitHub issue with the bug report template.
